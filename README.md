## 📚 Course 1: Foundations of Cybersecurity
*Status: In Progress*

### Key Concepts Learned:
*   **The CIA Triad:** 
    *   *Confidentiality:* Keeping data private using access controls and encryption.
    *   *Integrity:* Ensuring data is accurate and unaltered using hashing (like SHA-256).
    *   *Availability:* Making sure authorized users have constant access to systems and data.
*   **Security Frameworks:** Introduction to how organizations use frameworks (like NIST and CIS) to mitigate risk and protect assets.
*   **The 8 CISSP Domains:** Understanding the broad scope of security operations, from Asset Security to Risk Management.
